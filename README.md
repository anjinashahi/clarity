Clairty is expense tracking system. Users can add, edit, delete transaction, make categories, export data to csv.

Main Features
- Add incomes and expense
- Edit and delete the data
- View all list of incomes and expenses
- Dashboard overview
- Export to csv

  Tech Stack
  Frontend: React Typescript
  Backend: Node.js express,js
  Database: MondoDB Atlas
  Authentication: JWT

  Installation process
  Clone the repo

  Backend setup
  cd backend
  npm install
  npm run dev

  Frontend setup
  cd frontend
  npm install
  npm run dev

  Make an env file in backend and port 500, mongo uri and jwt scret
